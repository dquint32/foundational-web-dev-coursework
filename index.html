Purpose of This Site / Propósito del Sitio

This site documents the full set of projects completed for CIS‑3030: Business Web Page Development at MSU Denver. It serves as a technical archive demonstrating my foundational training in:

Semantic HTML5, modern CSS (Grid, Flexbox, responsive design), multimedia integration, JavaScript fundamentals (arrays, loops, and conditional logic), and accessibility and standards‑based development.

Although my professional portfolio highlights client‑ready, real‑world projects, this archive provides transparency into the academic foundations that support my work as a bilingual web developer.
