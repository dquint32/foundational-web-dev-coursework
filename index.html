Project Mission & Purpose

This digital archive documents my technical progression through the academic and technical foundations of web development. For detailed inquiries or professional services, please visit davidquintana.dev. This site serves as a transparent record of my proficiency in:

  • Semantic HTML5: Building meaningful, accessible document structures.
  • Modern CSS: Implementing responsive design via Flexbox, CSS Grid, and mobile-first strategies.
  • JavaScript Fundamentals: Developing logic through arrays, loops, and conditional statements.
  • Web Standards: Ensuring cross-browser compatibility and ADA accessibility compliance.

Technical Project Gallery

The following modules represent a semester-long deep dive into the New Perspectives on HTML5 and CSS curriculum. Each project demonstrates a shift from static markup to complex, interactive applications.

Tutorial 1: Core HTML5 Fundamentals

  • Curbside Thai Challenge: Implemented foundational HTML5 structure and semantic markup for a mobile food vendor website.
  • Menu: Curbside Thai Menu
  • Reviews: Curbside Thai Reviews

Tutorial 2: Core CSS Foundations

  • Tri & Succeed Sports Layout: Created a multi-page site using external CSS to control global typography and page regions.
  • TSS Coaches Page: Formatted navigation menus with dynamic hover effects and integrated custom web fonts.
  • Civil War History Site: Applied float-based layouts and dual navigation systems for a complex historical archive.

Tutorial 3: Designing a Page Layout

  • Pandaisia Chocolates: Implemented multi-column layouts using percentage-based widths and absolute positioning.
  • Pandaisia Monthly Specials: Engineered a grid-based product gallery and a scrollable awards sidebar.
  • Slate & Pencil Tutoring: Developed a four-column CSS Grid layout featuring elements that span multiple rows and columns.

Tutorial 4: Graphic Design with CSS

  • The Komatsu Family: Used RGBA color values and multiple background images to create visual depth without heavy image assets.
  • The Ferris Family: Utilized the border-image property and CSS filters (sepia) to evoke a historical aesthetic.
  • Chupacabra Music Fest: Engineered a functional 3D rotating cube using translateZ and rotate transforms in a 3D coordinate system.

Tutorial 5: Responsive Design and Accessibility

  • Trusted Friends Pre-K: Implemented a mobile-first strategy and a specialized print stylesheet that optimizes typography for physical documents.
  • Trusted Friends Parenting Tips: Developed nested flexbox layouts and integrated the attr(href) CSS function for printed URLs.
  • Cauli-Wood Gallery: Swapped background images specifically for mobile users to improve outdoor readability.

Tutorial 6: Working with Tables and Columns

  • DLR Nightly Schedule: Used <thead>, <tbody>, and <tfoot> tags with column grouping to manage diverse broadcast data.
  • DLR Morning Schedule: Utilized the CSS Multi-column Layout Module to flow descriptive text into a readable three-column format.
  • Marlin Internet: Integrated custom @font-face rules and multi-stop linear gradients to create a high-tech pricing interface.

Tutorial 7: Designing a Web Form

  • ACGIP Conference Registration: Applied custom regex patterns for validation and a three-state visual feedback system using :valid and :invalid pseudo-classes.
  • Red Ball Pizza Order Forms: Engineered a multi-page form flow with dynamic field state management based on user delivery selections.
  • Red Ball Pizza Survey: Deployed range sliders and multi-select dropdown lists organized within semantic <fieldset> containers.

Tutorial 8: Enhancing a Website with Advanced CSS

  • Frustrated Fox Game: Engineered complex sprite sheet animations using the steps() timing function and custom keyframe sequences.
  • Cinema Penguin: Fred Astaire Biography: Integrated HTML5 audio/video with custom WebVTT caption styling and an infinite marquee animation.
  • Cinema Penguin: Royal Wedding: Synchronized video playback with a 360-degree CSS rotation animation using checkbox-controlled triggers.

Tutorial 9: Getting Started with JavaScript

  • Bridger College Student Union: Developed a dynamic calendar using a seven-case switch statement and locale-aware date formatting.
  • Tulsa Summer Party Countdown: Implemented setInterval and Math logic to perform real-time millisecond-to-unit conversions for a holiday countdown.
  • Tulsa New Year's Bash: Engineered a self-correcting target date calculation to ensure countdowns always display future events.

Tutorial 10: Exploring Arrays, Loops, and Conditionals

  • VoterWeb Election Results: Processed multidimensional arrays of candidate data to generate interactive tables with visual percentage bars.
  • Lyman Hall Theater Event Listings: Created a 14-day time window for event filtering by performing millisecond arithmetic on current Date objects.
  • Lyman Hall Interactive Calendar: Built a modular function architecture to dynamically generate monthly calendar grids, including leap year logic.

Capstone Project: Tienda Salvadoreña

A Bilingual E-Commerce Website for Denver's Oldest Salvadoran Store

This capstone project synthesizes all techniques learned throughout CIS 3030 to create a culturally authentic digital storefront.

  • Visit Tienda Salvadoreña Website
  • Bilingual Translation System: Custom lang-toggle.js with 200+ key-value pairs and localStorage persistence.
  • Sophisticated Design Tokens: Utilizes CSS variables for a national color palette and clamp() for fluid typography.
  • Advanced Layouts: Combines CSS Grid galleries with nested Flexbox containers for responsive card scaling.
  • Interactive UI: Features a JavaScript-driven product category filter and a custom image zoom modal system.
  • Accessibility & Standards: Achieved WCAG 2.1 Level AA compliance through ARIA landmarks, touch-target optimization, and reduced motion support.
